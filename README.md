# Kar:sm

## Routes
* Inscription utilisateur : `/user/register`
* Connexion utilisateur : `/user/login`
* Déconnexion utilisateur : `/user/logout`
* Profil utilisateur : `/user/profile/{id}`
* Charte : `/charte`
* FAQ : `/faq`
* Admin : `/admin`


## Technologies
* PHP 7.0
* Symfony 3
* SASS
* Foundation 6
